#### 变量

let：定义变量，表示变量不可被重复定义

const：定义常量，且这个常量也不可被重复定义

作用域：let定义的变量和const定义的常量，定义在函数或代码块内，属于局部作用域，函数外部无法使用此变量或常量；定义在函数或方法体外部，属于全局作用域，在任何函数或方法体中都可以访问

箭头函数=>

使用 => 代替function关键字来定义函数函数名(参数列表) => {方法体}

特点：

  1. 只有一个参数时，参数列表的括号可以省略，只写这个参数函数名 = 参数 => {方法体}

    //省略参数列表的小括号
let look=a=> {
	return a+1;
}

  2. 函数中没有其它语句，只有一个return，则方法体的大括号也可以省略，直接写返回结果函数名 = (参数列表) => 返回结果

    //省略方法体的大括号
let look=a=>a+1;

函数
1.参数的扩展/展开

获取多余的参数

当方法的实参多于形参时，可以在形参的参数列表最后追加一个参数... 参数名，这个参数是一个数组，表示获取参数列表中剩余的参数，然后存到这个数组中，参数名可以自定义，但是这个数组必须写在形参列表的最后一个
function a (a,b,c, ...args) {}

获取数组

使用 ... 还可以获取数组中的值...数组名

var arr = [1,2,3,4,5,6];
console.log(...arr);

默认参数

在定义参数时，如果需要为这个参数指定默认值，直接参数名 = 默认值 即可

function look(a,b=10,c=20) {
	console.log(a,b,c);
}
look(1,2,3);

如果调用时没传入这个参数的值，则这个参数的值为默认值，如果传入了当前参数的值，则优先使用传入的值

解构赋值

解构赋值，一次为多个变量、对象或者数组赋值，使用解构赋值时，等号左右两侧的格式必须一样

示例：

var [{name,pwd},age,gender,[score1,score2,score13]] = [{name:"小明",pwd:"00000000"},18,"男",[60,77,84]];
console.log(name, pwd, age, gender, score1, score2, score13);

执行结果：小明 00000000 18 男 60 77 84
